# Memory Game - Android приложение 🎮

Мобильная игра "Найди пару" с поддержкой онлайн-режима через Firebase.

## 📋 Описание проекта

**Memory Game** - Android-приложение для игры в классическую игру "Найди пару". 
Приложение разработано в рамках выпускной квалификационной работы.

## 🚀 Режимы игры

### 1. Одиночная игра
Играйте против времени, открывайте пары карточек и набирайте очки.

### 2. Кооперативная игра
Играйте вдвоем на одном устройстве, соревнуйтесь кто найдет больше пар.

### 3. Онлайн-режим
Играйте с другом через интернет с двух разных устройств!

## 🛠 Технологии

- **Язык**: Kotlin
- **Минимальная версия Android**: API 24 (Android 7.0)
- **Целевая версия**: API 34
- **Архитектура**: MVVM
- **База данных**: Room (локальная) + Firebase Realtime Database (онлайн)
- **Аутентификация**: Firebase Anonymous Auth
- **Асинхронность**: Kotlin Coroutines + Flow
- **UI**: View Binding, RecyclerView, Material Design

## 🎮 Как играть в онлайн-режиме

### Игрок 1 (Хост):
1. Главное меню → Играть → Онлайн игра
2. Нажмите "Создать комнату"
3. Выберите уровень и режим таймера
4. Отправьте ID комнаты второму игроку
5. Дождитесь присоединения
6. Нажмите "Начать игру"

### Игрок 2 (Гость):
1. Главное меню → Играть → Онлайн игра
2. Найдите комнату хоста в списке
3. Нажмите на комнату для присоединения
4. Дождитесь начала игры
